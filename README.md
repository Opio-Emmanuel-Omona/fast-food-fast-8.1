## SETUP

### Requirements
```
- Node installed
```

### Installation and Setup

Run the following in your command line
```
- git clone https://github.com/Opio-Emmanuel-Omona/fast-food-fast-8.1.git
- cd fast-food-fast-8.1
- git checkout develop
- npm install
- npm run build
- npm start
```
